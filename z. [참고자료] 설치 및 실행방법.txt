  2. 터미널1에 명령어 입력 : node .\server\module\Server.js (리눅스일시, npm run linux-dev)
  3. 터미널2에 명령어 입력 : webpack --watch
  끝.

  1. 터미널(CMD, Powershell, 터미널 등 명령어창) 띄우기
  2. 프로그램 소스가 있는 위치로 이동 ex)cd /.../web-server
  3. 명령어 입력1 : webpack
  4. 명령어 입력2 : node .\server\module\Server.js
